proficiency_levels:
  - level: Awareness
    indicator: Clones repos; commits local changes; and resolves simple merge conflicts.
  - level: Basic
    indicator: Creates branches, submits pull requests, and reviews peers’ code.
  - level: Intermediate
    indicator: Designs branching strategies (e.g., GitFlow); automates CI triggers; manages pull-requests, tagging, and semantic versioning.
  - level: Advanced
    indicator: Defines branching strategies (GitFlow, trunk-based) and hooks; enforces code-review policies; secures repositories; integrates release pipelines.
  - level: Expert
    indicator: Architects enterprise-wide SCM governance and cross-repo dependency management.
