## INTERNAL ROUTING GUIDE (Sapience Use Only)

### Offering Routing Matrix

| If Client Indicates... | Route To... |
|------------------------|-------------|
| Leadership focus + manager or senior audience + multi-session | Rising Leaders Academy |
| Professional skills + individual contributors + series format | Professional Effectiveness Series |
| Self-mastery + any level + workshop format | Self-Mastery Workshop (B2B) |
| Organizational consulting + competency or talent systems | Competency Modeling Consulting |
| High customization + unique content needs | Custom Training Development |
| Pilot preference + limited budget | Pilot Workshop Engagement |
| Tech / cyber sector + leadership gaps | Technical Leadership Program |
| Healthcare sector + leadership gaps | Healthcare Leadership Program |

### Quality Gates

**Before moving Part A → Part B:**
- Budget aligns with likely offering scope
- Timeline is realistic
- Decision-maker identified
- Problem statement is training-addressable

**Before Part B → Proposal:**
- Competency priorities clear
- Success metrics defined
- Audience characteristics understood
- Constraints identified

**Before Part C → Design Phase:**
- Task analysis complete
- Learner profile documented
- Content audit done
- Assessment approach agreed
- Milestone plan confirmed
