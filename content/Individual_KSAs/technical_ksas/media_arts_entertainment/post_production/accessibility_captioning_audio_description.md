category: Technical
sector: media_arts_entertainment

  - "Accessibility"

description: >
  Creates closed captions (CEA-608/708, WebVTT) and audio description tracks meeting ITU-T H.702 and WCAG 2.2 media requirements.
source_frameworks:
  - "ITU-T H.702 Accessibility Recommendation"
  - "WCAG 2.2 Success Criteria"
proficiency_levels:
  - level: Awareness
    indicator: Differentiates CC vs subtitles; identifies audio-desc needs.
  - level: Basic
    indicator: Times captions; syncs to dialog; exports .srt and .vtt.
  - level: Intermediate
    indicator: Adds speaker IDs; embeds positional audio cues; meets minimum font legibility.
  - level: Advanced
    indicator: Automates caption via ASR then QC; mixes AD track; verifies compliance.
  - level: Expert
    indicator: Leads accessibility policy; integrates multi-lang captions with AI; mentors captioners.
