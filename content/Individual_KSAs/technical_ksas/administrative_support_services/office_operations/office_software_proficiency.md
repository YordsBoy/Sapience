ksa_id: office_software_proficiency
label: Office Software Proficiency
category: Technical
sector: Administrative_Support
horizon: core_2025
cluster_tags:
  - "Digital Skills"
  - "Office Software"
description: >
  Efficiently uses office productivity suites (word processing, spreadsheets,
  presentations, databases) and collaboration tools to create, analyse, and
  share information while adhering to accessibility and security best practices.
source_frameworks:
  - "EU DigComp 2.1 Framework"
proficiency_levels:
  - level: Awareness
    indicator: Opens, saves, and prints files.
  - level: Basic
    indicator: Uses formatting, formulas, and slide templates; shares docs via cloud links.
  - level: Intermediate
    indicator: Builds pivot tables, mail merges, slide masters; collaborates in real time.
  - level: Advanced
    indicator: Automates tasks with macros; troubleshoots file issues; ensures document accessibility.
  - level: Expert
    indicator: Trains teams; evaluates and deploys new productivity tools; enforces digital-skills standards.
