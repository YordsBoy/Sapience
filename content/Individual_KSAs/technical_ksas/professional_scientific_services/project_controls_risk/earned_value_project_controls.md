category: Technical
sector: professional_scientific_services
horizon: core
cluster_tags:
  - "Earned Value"
  - "Project Controls"
description: Implementing Earned Value Management (EVM) per ANSI/EIA-748 guidelines to integrate scope, schedule, and cost, calculate performance indices, and forecast Estimate at Completion (EAC) for scientific and consulting projects.
source_frameworks:
  - "ANSI/EIA-748-D Earned Value Management Systems Standard (U.S. standards body — ANSI / Electronic Industries Alliance; proprietary standard; conceptual alignment only)"
  - "NDIA Integrated Program Management Division Intent Guide (industry association — National Defense Industrial Association; publicly available)"
  - "PMI Practice Standard for Earned Value Management 2nd Edition (commercial professional association — Project Management Institute; conceptual alignment only)"
proficiency_levels:
  - level: Awareness
    indicator: Defines Planned Value (PV), Earned Value (EV), Actual Cost (AC); records baseline data.
  - level: Basic
    indicator: Updates weekly status; Sets work packages; updates EV; calculates Cost Performance Index (CPI); flags variances.
  - level: Intermediate
    indicator: Performs VAC analysis; Performs Estimate to Complete (ETC); integrates risk reserves; produces Control Account Plans (CAPs).
  - level: Advanced
    indicator: Implements EVM systems; conducts Integrated Baseline Reviews (IBRs); aligns EVM with agile metrics; integrates risk contingencies; meets EV surveillance.
  - level: Expert
    indicator: Leads corporate EVM governance; Certifies EVMS compliance; interfaces with government compliance audits; publishes project-controls research.
