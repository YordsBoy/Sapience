ksa_id: cloud_finops_cost_optimization
label: Cloud FinOps Cost Optimization
category: Technical
sector: core_it_cloud
horizon: emerging
cluster_tags:
  - "FinOps"
  - "Cost Management"
description: >
  Tracks, allocates, and optimizes cloud spend; sets KPIs (unit economics), rightsizes workloads, negotiates committed-use discounts, and drives culture change per FinOps Framework.
source_frameworks:
  - "FinOps Foundation – State of FinOps 2025 (nonprofit industry report; publicly available)"
  - "FinOps Foundation Practitioner Certification Guide (commercial certification body; conceptual alignment only)"
  - "AWS Cost Management White Paper (vendor documentation; conceptual alignment only — equivalent platforms may demonstrate competency)"
proficiency_levels:
  - level: Awareness
    indicator: Retrieves basic cost-and-usage reports; identifies untagged resources.
  - level: Basic
    indicator: Implements cost-allocation tags; sets budget alerts using cloud provider cost management tools (e.g., AWS Cost Explorer, Azure Cost Management, GCP Cost Management Console).
  - level: Intermediate
    indicator: Rightsizes instances; negotiates committed-use discount agreements (e.g., AWS Savings Plans, Azure Reserved Instances, GCP Committed Use Discounts); presents monthly show-back reports to stakeholders; implements autoscaling and spot-capacity.
  - level: Advanced
    indicator: Automates anomaly detection using Application Programming Interface (API) feeds; forecasts spend allocates show-back/charge-back; integrates cost KPIs into Continuous Integration pipelines.
  - level: Expert
    indicator: Leads organization-wide FinOps program; drives cloud-service provider contract optimisations; mentors cross-functional FinOps teams; influences vendor pricing.
