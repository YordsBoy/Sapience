proficiency_levels:
  - level: Awareness
    indicator: Performs basic CRUD operations and routine backups; executes SELECT queries; explains ACID properties and their significance for database reliability.
  - level: Basic
    indicator: Configures roles/permissions; normalises tables; sets indexes; backups; monitors query plans.
  - level: Intermediate
    indicator: Tunes queries; implements replication, sharding, and disaster-recovery strategies.
  - level: Advanced
    indicator: Tunes performance for high-throughput workloads and orchestrates migration/upgrade paths.
  - level: Expert
    indicator: Designs multi-region data-platform strategy, multi-modal data platforms, cost governance, and automated self-healing clusters.
