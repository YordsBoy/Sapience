  - level: Basic
    indicator: Calculates SCOR Level‑1 KPIs (perfect‑order fulfilment); defines process elements; documents value streams.
  - level: Intermediate
    indicator: Conducts gap analyses; runs what‑if simulations; aligns KPIs with cost‑to‑serve models.
  - level: Advanced
    indicator: Implements supply‑chain control towers; integrates real‑time data feeds; optimises cross‑functional trade‑offs.
  - level: Expert
    indicator: Leads global SCOR transformations; mentors practitioners; publishes supply‑chain strategy research.
