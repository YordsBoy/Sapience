  - level: Awareness
    indicator: Applies basic modules in a sandbox cloud account.
  - level: Basic
    indicator: Provisions and applies infrastructure stacks; stores state securely in a remote backend.
  - level: Intermediate
    indicator: Builds reusable modules, CI validation, policy-as-code.
  - level: Advanced
    indicator: Manages multi-cloud environments and provider accounts, drift detection, and orchestration across workspaces or equivalent state-isolation constructs.
  - level: Expert
    indicator: Sets enterprise IaC standards; integrates with FinOps/SecOps.
