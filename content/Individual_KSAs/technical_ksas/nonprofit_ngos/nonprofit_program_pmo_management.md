label: Nonprofit Program & PMO Management
category: Technical
sector: Nonprofit_NGO
horizon: core_2025
cluster_tags: ["Program Management","PMO","Nonprofit"]
description: Applying Project Management Institute (PMI) methodologies and adaptive approaches to design, execute, and monitor nonprofit initiatives, aligning scope, budget, timeline, and impact metrics within a Program Management Office (PMO) structure.
source_frameworks: ["PMI PMD Pro Guide","PMI Standard for Program Management 4th Ed."]
proficiency_levels:
  - level: Awareness
    indicator: Defines project vs. program; maintains task lists; tracks basic costs.
  - level: Basic
    indicator: Develops Logical Framework Approach (LFA) matrices; manages risk registers; updates Gantt charts.
  - level: Intermediate
    indicator: Implements adaptive iteration cycles; aligns key performance indicators (KPIs) to donor agreements; facilitates lessons‑learned reviews.
  - level: Advanced
    indicator: Establishes PMO governance; integrates agile and results‑based management (RBM); optimises resource allocation across programs.
  - level: Expert
    indicator: Directs multi‑country program portfolios; shapes sector PM standards; publishes nonprofit PM research.
