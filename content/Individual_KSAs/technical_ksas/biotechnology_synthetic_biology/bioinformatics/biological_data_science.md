description: >
  Analyses multi-omics datasets (genomic, transcriptomic, proteomic) using computational pipelines; applies statistics and machine-learning to uncover biological insight and guide
  experimental design.

  - "NIH Bioinformatics Core Competencies"
  - "ELIXIR Bioinformatics Core Competencies"
  - "FAIR Data Principles"
  - "GA4GH Standards"
proficiency_levels:
  - level: Awareness
    indicator: Performs FASTQ quality checks; runs BLAST; interprets basic heatmaps.
  - level: Basic
    indicator: Aligns reads with BWA; quantifies expression (salmon); manages version-controlled pipelines; executes canned RNA-seq or variant-calling workflows.
  - level: Intermediate
    indicator: Conducts differential‑expression analysis; visualises networks; applies metadata standards; customises scripts; integrates datasets; performs statistical tests.
  - level: Advanced
    indicator: Automates multi‑omics workflows; builds scalable pipelines on HPC/cloud; applies ML for pattern discovery; implements cloud computing; ensures FAIR compliance.
  - level: Expert
    indicator: Leads data‑science core; develops new computational methods; leads large-scale consortium analyses; publishes novel bioinformatics tools; shapes GA4GH policies.
