    indicator: Interfaces digital/analog sensors; debounces inputs; sends data via serial or Wi‑Fi; solders headers; reads sensor data; logs serial output.
  - level: Intermediate
    indicator: Publishes data to cloud via MQTT; implements low-power sleep; designs simple printed‑circuit boards (PCBs); utilizes interrupt routines.
  - level: Advanced
    indicator: Optimises power consumption; secures communication with Transport Layer Security (TLS); tests electromagnetic compatibility (EMC).
  - level: Expert
    indicator: Leads maker‑to‑market IoT productisation; certifies devices for safety; mentors community electronics labs.
