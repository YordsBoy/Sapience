proficiency_levels:
  - level: Awareness
    indicator: Maps attack surface; recognizes default credentials risk.
  - level: Basic
    indicator: Implements password policy; disables unused ports; patches firmware.
  - level: Intermediate
    indicator: Segments OT-IT networks; deploys monitoring; logs events.
  - level: Advanced
    indicator: Runs vulnerability scans; creates incident-response plan; trains vendors.
  - level: Expert
    indicator: Leads portfolio cyber-program; mentors operators; aligns to industry cyber-maturity frameworks (e.g., RECC maturity model).
