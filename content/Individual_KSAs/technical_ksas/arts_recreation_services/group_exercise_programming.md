proficiency_levels:
- level: Awareness
indicator: Demonstrates basic choreography; cues tempo changes; checks equipment for safe use.
- level: Basic
indicator: Structures class phases (warm‑up, work, cool‑down); selects playlists matching target heart‑rate zones.
- level: Intermediate
indicator: Utilises periodisation across class cycles; adapts intensity for varied fitness levels; monitors Class Average Heart Rate (CAHR).
- level: Advanced
indicator: Designs signature formats; integrates wearable tracking; evaluates participant retention metrics.
- level: Expert
indicator: Trains instructors; licenses programme formats internationally; publishes research on group‑exercise adherence.
