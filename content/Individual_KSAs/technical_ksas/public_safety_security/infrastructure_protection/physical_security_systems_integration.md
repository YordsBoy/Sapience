  - "UL 294 Access Control Standard"
  - "ONVIF Profile S/G"

proficiency_levels:
  - level: Awareness
    indicator: Identifies security components; describes server vs cloud VMS; interprets basic floor plans; follows badge protocols.
  - level: Basic
    indicator: Installs CCTV cameras; configures door controllers; sets cardholder roles; tests alarm signals.
  - level: Intermediate
    indicator: Integrates systems via PSIM (Physical Security Information Management); conducts vulnerability assessments; analyses video analytics; implements cybersecurity hardening.
  - level: Advanced
    indicator: Designs converged IT‑OT security; implements zero‑trust access; manages maintenance KPIs; performs throughput calculation; ensures fail-over; aligns with SAFETY-Act certification.
  - level: Expert
    indicator: Architects city-scale PSIM; oversees enterprise security integration; adheres to CPTED (Crime Prevention Through Environmental Design); publishes physical‑security design guides.
