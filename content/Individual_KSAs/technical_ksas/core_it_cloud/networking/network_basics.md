proficiency_levels:
  - level: Awareness
    indicator: Identifies common protocols (HTTP, DNS, SSH); identifies IP vs. DNS issues using ping or traceroute.
  - level: Basic
    indicator: Configures subnets, VLANs, and basic firewall rules; tests connectivity.
  - level: Intermediate
    indicator: Diagnoses latency/packet-loss with traceroute, tcpdump; analyzes packet captures, optimizes routing tables, and implements VPNs.
  - level: Advanced
    indicator: Designs VLANs, VPNs, BGP peering; and redundant, secure network topologies for cloud/hybrid environments.
  - level: Expert
    indicator: Leads enterprise network architecture, zero-trust segmentation, and high-availability strategies.
