  - level: Awareness
    indicator: Defines Planned Value (PV), Earned Value (EV), Actual Cost (AC); records baseline data.
  - level: Basic
    indicator: Updates weekly status; calculates Cost Performance Index (CPI); flags variances.
  - level: Intermediate
    indicator: Performs Estimate to Complete (ETC); integrates risk reserves; produces Control Account Plans (CAPs).
  - level: Advanced
    indicator: Implements EVM systems; conducts Integrated Baseline Reviews (IBRs); aligns EVM with agile metrics.
  - level: Expert
    indicator: Leads corporate EVM governance; interfaces with government compliance audits; publishes project‑controls research.
